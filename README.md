# ✦ موقع كوتش الاء محمد

موقع شخصي لكوتش الاء محمد — مدربة معتمدة بالأسواق المالية وريادة الأعمال.

## 🚀 التشغيل

```bash
npm install
npm run dev
```

## 📁 هيكل الملفات

```
├── index.html          # الصفحة الرئيسية
├── SbWebapp.jsx        # مكون React الرئيسي
├── package.json        # ملف المشروع
├── vite.config.js      # إعدادات Vite
└── public/
    └── images/         # صور الموقع (أضف صورة البروفايل هنا)
```
